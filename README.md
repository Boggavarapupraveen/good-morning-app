# Good Morning Full Stack Test

## Tech Stack
- React JS
- Node JS (Express)
- PostgreSQL
- Flutter

## Features
- Web and Mobile form
- Stores name, phone, email
- PostgreSQL database

## How to Run
1. Start PostgreSQL
2. Run backend: `node index.js`
3. Run frontend: `npm start`
4. Run Flutter app

## Core Strengths
- Full Stack Development
- REST APIs
- Database Design
- Mobile App Development
